### Getting started

1. git clone 
2. cd flask-socialWeb
3. pip install -r requirement.txt
4. python app.py


### Deploy to heroku

[heroku-flask-deploy(https://devcenter.heroku.com/articles/getting-started-with-python#deploy-the-app)

1. git push heroku master
2. heroku ps:scale web=1
3. heroku open
